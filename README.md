# Quaderno Go Client

An API client for [Quaderno.io](https://quaderno.io) in Go.

This is an unofficial library, so it should be used at your own risk. However, we actively use it in our production
applications, so the supported endpoints are stable.

If you encounter a bug, please [open an issue](../../issues).

Feature requests are welcome, but we cannot guarantee they will be prioritised if they don't align with our current
usage of the API. If you need an urgent feature or endpoint added, submit a PR.
